== Description ==

Patrika is best suited for news & magazine sites. It is quite easy to set up a site for news portals, online magazines, newspaper, bloggers and content publishers. With a modern design and mobile friendly features this theme is also coded to make the full benefits of SEO.

For more information about Patrika theme please go to https://www.themesmandu.com/product/patrika/.

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type Patrika in the search form and press the 'Enter' key on your keyboard.
3. Click on the 'Activate' button to use your patrika theme right away.
4. Go to https://www.themesmandu.com/product/patrika/ for a guide on how to customize this theme.
5. Navigate to Appearance > Customize in your admin panel and customize the theme to your taste.
